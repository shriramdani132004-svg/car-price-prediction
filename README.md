# 🚗 Car Price Prediction App

This project predicts car prices using Machine Learning and provides an interactive web interface using Streamlit.

---

## 📌 Features

- Predict car price based on specifications
- Machine Learning model (Random Forest)
- Data preprocessing and feature engineering
- Streamlit web application

---

## 🧠 Technologies Used

- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

---

## ⚙️ How It Works

1. Data is cleaned and preprocessed
2. Feature engineering is applied
3. Model is trained using Random Forest
4. Predictions are made via Streamlit UI

---

## ▶️ Run the App

```bash
streamlit run app.py
